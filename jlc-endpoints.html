🔐 How to authenticate
1. Call POST /signin → receive accessToken
2. Store it securely (never in localStorage)
3. Attach to every protected request: Authorization: Bearer <accessToken>
4. For Gamification endpoints marked cognitoId in body — decode the JWT locally to extract the sub claim and pass it as cognitoId in the request body. No Authorization header needed for those.
